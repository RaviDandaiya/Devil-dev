# Plumber Platformer

A vibrant, classic 2D side-scrolling platformer built with Vanilla JavaScript and HTML5 Canvas.

## Features
- **Classic Platforming**: Run, jump, and stomp your way through levels.
- **Scrolling Camera**: Smooth camera tracking for an immersive experience.
- **Mystery Blocks**: Hit '?' blocks to discover coins and size-increasing power-ups.
- **Enemies**: Beware of patrolling foesâ€”stomp them to survive!
- **Level Completion**: Reach the golden flag to secure victory.

## Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/) (for development server)

### Running Locally
1.  Clone the repository (once live).
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Start the development server:
    ```bash
    npm run dev
    ```
4.  Open your browser to the provided local URL (usually `http://localhost:5173`).

### Controls
- **Arrow Left / Right**: Move
- **Spacebar**: Jump
- **Mouse Click**: Start Game (on overlay)

## Tech Stack
- **Frontend**: Vanilla JavaScript, HTML5 Canvas, CSS3
- **Build Tool**: Vite
